BLS Products

Our products are revolutionizing the industry standard.

Mail Bailer

Convert emails to PDFs without the hassle Legal investigations routinely involve hundreds of thousands of electronic documents, which includes emails. The subsequent e-discovery process is resource intensive and a tremendous burden for any Legal Team. MailBailer is an automated email search and analysis tool designed to simplify the e-discovery process.

Features:
  • Covert EML and PST files to PDFs
  • Extract all attachments from PST or EML files
  • Apply BATES numbering to converted PDFs
  • Search through emails to discover and highlights text strings

Splinternet

Execute continuous attacks for up to the minute detection and prevention. Breach and Attack simulation (BAS) provides for continuous validation and improvement of network-, host-, email-, and web-based security controls. SplinterNet is a BAS platform that provides automated, playbook-based attack execution for the validation of enterprise security controls. SplinterNet provides a custom command and control (C2) framework with multiple implants that can be operated by Offensive and Defensive personnel in order to tune security controls and train Security Operations Center (SOC) analysts.

Features
  • Browse and execute automated attacks based on MITRE ATT&CK or security control categories
  • Quickly simulate new or existing threats
  • Fully customizable covert communication channels
  • Automate attacks for all phases of the kill chain
  • Gather and examine defensive artifacts to easily create post exercise out-briefs
